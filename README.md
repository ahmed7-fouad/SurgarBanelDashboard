# 🚀 React Frontend Internship Task

## 📌 Overview

This project is a responsive frontend application built using modern React practices.
It includes authentication flow, dashboard, list, and details pages with API integration.

---

## 🛠️ Tech Stack

* React (Vite)
* React Router (Data Router)
* Bootstrap / React Bootstrap
* Axios
* Docker & Docker Compose
* Recharts
* MUI For Icons




---

## 📄 Features

* Login Page (Fake Authentication)
* Dashboard Page
* Users List Page (API Integration)
* User Details Page
* Responsive Design (Mobile & Desktop)

---
